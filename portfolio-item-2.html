Student Management System in Python

The Student Database Management System is an application software which helps in maintaining the student records in a school or college.

The student data is stored in the database and it can be accessed easily whenever needed. The application has features such as adding a new student, displaying all the students' data, printing records of a student, deleting or updating a record, etc.

The software is devloped in Python. Various libraries have been used for the UI and other functions.
